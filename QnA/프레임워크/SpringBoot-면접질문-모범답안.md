# Spring Boot 면접질문-모범답안

## 면접 질문
- Spring Boot를 선택한 이유는?
- 자동설정은 어떻게 동작하나요?
- 3.5.13 사용 시 운영에서 중요하게 보는 포인트는?

## 모범 답안
<details>
  <summary>Spring Boot를 선택한 이유는?</summary>

- 결론: 빠른 개발 생산성과 안정적인 생태계 때문에 선택했습니다.
- 근거: starter/autoconfiguration/actuator로 개발-운영 표준화를 빠르게 구축할 수 있습니다.
- 트레이드오프: 내부 동작 이해 없이 쓰면 장애 분석이 어려워집니다.

</details>

<details>
  <summary>자동설정은 어떻게 동작하나요?</summary>

- 결론: 클래스패스/프로퍼티/빈 조건에 따라 필요한 빈을 자동 등록합니다.
- 근거: `@Conditional` 계열 조건과 auto-config 후보가 핵심 메커니즘입니다.
- 트레이드오프: 편리하지만 의도치 않은 빈 등록이 발생할 수 있어 조건 리포트 확인이 필요합니다.

</details>

<details>
  <summary>3.5.13 사용 시 운영에서 중요하게 보는 포인트는?</summary>

- 결론: 관측(메트릭/헬스), DB 풀, 타임아웃 정책, 배포 회귀 검증을 우선 관리합니다.
- 근거: 실제 장애는 프레임워크 자체보다 설정/연계 경계에서 자주 발생합니다.
- 트레이드오프: 보수적 설정은 안정성을 높이지만 최대 처리량이 줄 수 있습니다.

</details>
