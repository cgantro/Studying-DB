# AGENTS.md - QnA 하위 지침

적용 범위: `QnA/`

## 목적
- 실제 면접 발화 가능 답변을 만든다.

## 답변 구조
- `질문 의도 -> 핵심 답변(한 줄) -> 근거 경험 -> 결과/교훈 -> 꼬리질문 대비`
- 한 답변은 짧고 명확하게 작성한다.

## 분리 원칙
- 기술 사실 원문은 `CS공부/`에 둔다.
- QnA에는 말하기 버전과 연결 링크만 둔다.

## 인성 답변
- 주장 대신 행동/판단 기준/결과를 포함한다.
- 팀 성과와 개인 기여를 분리한다.

## 상태/근거
- 진행중/미완성 프로젝트를 완료형으로 말하지 않는다.
- 수치가 불명확하면 `확인 필요`를 붙인다.
