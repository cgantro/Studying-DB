# 기업 분석 요약

## 핵심 포인트
- B2B 중심: 안정성/운영성/보안/유지보수성 비중 큼
- AI/클라우드/업무자동화 축에서 제품/플랫폼 확장
- 고객 업무 맥락 이해 + 시스템 통합 능력이 중요

## 제품/솔루션(요약, 최신성 확인 필요)
- AI 에이전트/생성형 AI 플랫폼: `FabriX` (기업 데이터 통합/활용 기반으로 에이전트 설계/운영을 지원하는 방향)
- 협업/업무 생산성: `Brity` 계열(메일/메신저/미팅/문서 흐름 + AI 기능 결합)
- 업무 자동화: `Brity Automation` (RPA 포함 End-to-End 자동화 방향)
- 주의: 세부 기능/수치(정확도 %, 라이브러리 개수 등)는 **마케팅 자료 기반일 수 있어 면접에서 숫자 단정은 보류**하고, 질문이 나오면 “확인 후 공유”로 대응
- 상세 정리: [05_제품-솔루션-분석.md](05_제품-솔루션-분석.md)

## 면접 시 강조
- 기술 깊이 + 운영 현실 감각
- 단일 기능보다 시스템 경계/장애 대응
- 협업 프로세스 개선 경험

## 내 경험 연결(요약)
- 협업 솔루션/플랫폼 관점: “여러 사용자가 동시에 쓰는 환경에서 상태/권한/일관성을 유지”하는 문제를 좋아하고, 프로젝트에서도 실시간 데이터 흐름에서 병목과 상태 불일치를 줄이는 쪽을 맡아왔다.
- 근거 프로젝트:
  - RobotPal: 카메라 스트리밍 파이프라인 병목 측정/개선(측정 조건 문서 존재)
  - Autowing Car: MQTT↔WebSocket 실시간 흐름 + 인증/배포까지 E2E로 구성
