# 온라인 합주 플랫폼 (TempoLink)

## 한 줄 정의
실시간 합주를 위한 저지연 오디오/세션 동기화 중심의 플랫폼 설계 프로젝트.

## 현재 상태
- **미완성**
- 기획/설계 단계
- 기술 스택 후보: C++20, Java, Spring Boot

## 설계 방향
- 오디오 경로와 제어 경로 분리
- 세션 단위 상태 관리
- 지연/동기화 품질 지표 우선 설계

## 예상 CS 연결
- 네트워크: UDP/TCP 분리 전략, 패킷 손실 보정
- 운영체제: 실시간 스레드/버퍼링
- 데이터베이스: 세션 메타데이터/이력 분리 저장
