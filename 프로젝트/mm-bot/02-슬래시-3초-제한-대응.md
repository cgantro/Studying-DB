# mm-bot - Slash Command 3초 제한 대응

## 문제
- Mattermost Slash Command는 응답을 3초 안에 반환해야 한다.
- RAG(검색) + LLM 호출은 네트워크/모델 상황에 따라 더 걸린다.

## 해결
- Webhook 수신 즉시 “처리 중...” 응답을 반환한다.
- 별도 단계에서 Mattermost REST API로 최종 답변을 채널에 POST한다.

## 대안
- 전용 웹 UI에서 장기 요청 허용:
  - 장점: 제한이 덜함
  - 단점: 사용자 채널(Mattermost)에서 바로 쓰는 UX가 깨짐
