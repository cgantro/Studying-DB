# AGENTS.md - CS공부 하위 지침

적용 범위: `CS공부/`

## 목적
- 지식 원본 품질을 높이는 것이 목적이다.
- 답변체보다 개념 정확도와 구조적 설명을 우선한다.

## 문서 구조
- 기본 순서: `개념 -> 왜 필요한가 -> 동작 원리 -> 장단점 -> 한계 -> 대안 -> 프로젝트 연결`
- 실습 문서가 가능하면 `예제 코드 -> 실행/관측 결과 -> 해석`을 포함한다.

## 금지
- 면접 답변 문장 중심으로 작성 금지
- 근거 없는 성능 비교 금지

## 마무리 체크
- 개념 오해 가능성이 있는 용어 정의를 넣었는가
- 실제 프로젝트 연결 링크를 넣었는가
- 한계와 대안을 빠뜨리지 않았는가
