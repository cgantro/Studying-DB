- 네트워크는 컴퓨터 간 통신 규칙과 연결 흐름을 다룬다.

## 더 자세한 설명
- 네트워크는 컴퓨터끼리 데이터를 주고받는 규칙과 경로를 다루는 분야다.
- 주소(IP), 포트, 프로토콜, 라우팅 같은 개념이 함께 움직여야 통신이 가능하다.
- 웹 요청 하나가 나가기까지 DNS 조회, TCP 연결, HTTP 요청, 응답 수신 같은 과정이 이어진다.
- 네트워크를 이해하면 "어디서 끊겼는지"를 단계별로 추적할 수 있다.
