# 웹(HTTP / 인증 / CORS / DNS)

## 한 줄 정의
웹은 **브라우저/클라이언트가 DNS로 서버를 찾고, TCP/TLS 위에서 HTTP로 리소스를 주고받는 전체 흐름**을 의미한다.

## 왜 배우는가
- “도메인은 맞는데 접속이 안 됨”, “로그인은 되는데 브라우저에서만 실패” 같은 실무 이슈는 HTTP/CORS/DNS가 합쳐져 발생한다.
- 인증(쿠키/JWT)과 CORS, 캐시, 프록시(Nginx)까지 한 번에 연결해서 설명할 수 있어야 장애 분석 속도가 빨라진다.
