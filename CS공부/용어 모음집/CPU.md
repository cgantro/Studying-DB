# CPU

## 정의
Central Processing Unit. 메모리의 명령어를 가져와 해석하고 실행하는 중앙 처리 장치다.

## 사용 환경
프로그램의 명령어를 실제로 실행하는 모든 상황에서 중심이 된다. 애플리케이션 코드, 운영체제 커널 코드, 인터럽트 처리 루틴 모두 CPU 위에서 실행된다.

## 사용 조건
명령어와 데이터가 메모리 계층에서 공급되어야 하고, PC와 레지스터 같은 실행 상태가 올바르게 유지되어야 한다.

## 예시
프로그램의 덧셈, 비교, 분기, 메모리 접근은 결국 CPU 명령어 흐름으로 실행된다.

## 주의점
- CPU 성능은 클럭만으로 결정되지 않는다.
- 메모리 접근, 캐시 미스, 분기 예측 실패도 실행 시간을 크게 바꾼다.
- 운영체제 문맥 교환은 CPU 상태 저장과 복원 비용을 만든다.
