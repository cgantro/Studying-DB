# TLB

## 정의
최근 주소 변환 결과를 저장하는 빠른 캐시다.

## 사용 환경
가상 주소 변환을 빠르게 하기 위해 CPU의 메모리 관리 장치 근처에서 사용된다.

## 사용 조건
최근 사용한 가상 페이지 번호와 물리 프레임 번호 매핑이 TLB에 들어 있어야 hit가 발생한다.

## 예시
TLB hit가 나면 페이지 테이블을 직접 조회하지 않아도 된다.

## 주의점
- TLB는 데이터 캐시가 아니라 주소 변환 캐시다.
- 문맥 교환이나 페이지 테이블 변경 시 TLB 무효화가 필요할 수 있다.
- TLB hit율이 낮으면 메모리 접근 비용이 커진다.
