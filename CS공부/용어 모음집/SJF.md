# SJF

## 정의
Shortest Job First. 실행 시간이 짧을 것으로 예상되는 작업을 먼저 실행하는 방식이다.

## 사용 환경
여러 실행 가능한 작업 중 어떤 작업에 CPU를 줄지 결정하고 평가할 때 사용된다.

## 사용 조건
준비 큐, 우선순위, 예상 실행 시간, time quantum 같은 정책 입력이 필요하다.

## 예시
평균 대기 시간을 줄일 수 있지만 긴 작업이 굶을 수 있다.

## 주의점
- 평균 대기 시간을 줄이는 정책이 항상 공정한 것은 아니다.
- 응답성과 처리량은 서로 충돌할 수 있다.
- 우선순위 정책은 starvation을 만들 수 있어 aging 같은 보완책이 필요하다.
