# 페이지 테이블

## 정의
가상 페이지 번호를 물리 프레임 번호로 변환하는 표다.

## 사용 환경
가상 메모리를 사용하는 운영체제에서 가상 주소를 물리 주소로 변환할 때 사용된다. 프로세스마다 별도의 주소 공간을 제공하는 핵심 구조다.

## 사용 조건
페이지 크기, 가상 페이지 번호, 물리 프레임 번호, valid bit와 권한 정보가 관리되어야 한다.

## 예시
가상 페이지 5가 물리 프레임 20에 있다는 정보를 가진다.

## 주의점
- 페이지 테이블 조회는 비용이 있으므로 TLB가 함께 사용된다.
- TLB miss와 페이지 폴트는 다른 개념이다.
- 프로세스가 바뀌면 주소 공간도 바뀌므로 페이지 테이블 기준도 달라질 수 있다.
