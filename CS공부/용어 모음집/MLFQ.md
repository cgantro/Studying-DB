# MLFQ

## 정의
MLFQ(Multi-Level Feedback Queue)는 여러 우선순위 큐를 두고 작업의 실행 패턴에 따라 큐를 올리거나 내리는 스케줄링 방식이다.

## 사용 환경
대화형 작업은 빠르게 반응시키고, CPU를 오래 쓰는 작업은 낮은 우선순위로 보내고 싶을 때 사용된다.

## 사용 조건
여러 ready queue, 큐별 time quantum, 우선순위 승격/강등 규칙, starvation 방지 규칙이 필요하다.

## 예시
사용자 입력 처리처럼 CPU를 짧게 쓰고 자주 I/O 대기를 하는 작업은 높은 큐에 남기 쉽다. 반대로 긴 계산 작업은 CPU를 오래 쓰므로 낮은 큐로 내려갈 수 있다.

## 주의점
- MLFQ는 단순한 Priority Scheduling보다 동적이다.
- 규칙을 잘못 정하면 특정 작업이 계속 낮은 큐에 머물 수 있다.
- starvation 방지를 위해 오래 기다린 작업을 다시 높은 큐로 올리는 규칙이 필요하다.
