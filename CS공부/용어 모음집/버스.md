# 버스

## 정의
CPU, 메모리, 입출력 장치 사이에 데이터와 주소, 제어 신호가 이동하는 통로다.

## 사용 환경
CPU가 디스크, 네트워크, 키보드 같은 외부 장치와 통신할 때 사용된다.

## 사용 조건
장치 제어 레지스터, 드라이버, 인터럽트 라인 또는 DMA 설정이 올바르게 준비되어야 한다.

## 예시
CPU가 메모리를 읽을 때 주소 버스와 데이터 버스를 통해 값이 이동한다.

## 주의점
- 폴링은 단순하지만 CPU 시간을 낭비할 수 있다.
- 인터럽트는 효율적이지만 너무 잦으면 오버헤드가 된다.
- DMA는 CPU 부담을 줄이지만 메모리 일관성과 권한 관리가 필요하다.
