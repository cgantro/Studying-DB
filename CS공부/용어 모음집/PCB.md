# PCB

## 정의
Process Control Block. 프로세스 상태와 자원 정보를 저장하는 커널 자료구조다.

## 사용 환경
운영체제가 실행 단위를 만들고, CPU 시간을 나눠 주고, 실행 상태를 저장할 때 사용된다.

## 사용 조건
스케줄러가 실행 대상을 선택하고, 필요한 레지스터/스택/주소 공간 정보가 저장되어 있어야 한다.

## 예시
문맥 교환 때 CPU 레지스터 값과 상태가 PCB에 저장된다.

## 주의점
- 프로세스는 자원 소유 단위, 스레드는 실행 흐름 단위로 보는 것이 좋다.
- 스레드는 메모리를 공유하므로 동기화 문제가 생긴다.
- 문맥 교환은 너무 잦으면 성능 저하를 만든다.
