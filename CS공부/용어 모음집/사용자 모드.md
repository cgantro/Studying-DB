# 사용자 모드

## 정의
일반 프로그램이 제한된 권한으로 실행되는 CPU 모드다.

## 사용 환경
사용자 프로그램이 파일, 프로세스, 메모리, 장치 같은 보호된 자원을 요청할 때 사용된다.

## 사용 조건
사용자 모드에서 직접 수행할 수 없는 작업은 정해진 시스템 콜 인터페이스를 통해 커널 모드로 진입해야 한다.

## 예시
사용자 모드 프로그램은 하드웨어를 직접 제어하지 못하고 시스템 콜을 사용한다.

## 주의점
- 시스템 콜은 일반 함수 호출보다 비용이 크다.
- 커널 모드 코드는 시스템 전체에 영향을 줄 수 있어 엄격히 보호된다.
- 트랩과 인터럽트는 발생 원인과 동기성에서 구분된다.
