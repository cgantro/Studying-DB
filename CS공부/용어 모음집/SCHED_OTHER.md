# SCHED_OTHER

## 정의
SCHED_OTHER는 Linux에서 일반 작업에 사용하는 기본 시분할 스케줄링 정책이다. 커널 내부에서는 SCHED_NORMAL이라고도 부른다.

## 사용 환경
일반 애플리케이션, 서버 프로세스, 셸에서 실행하는 대부분의 프로세스처럼 특별한 실시간 보장이 필요 없는 작업에 사용된다.

## 사용 조건
일반적으로 static priority 0에서 동작하며, nice 값 같은 정보가 CPU 배분에 영향을 준다. 실시간 정책의 runnable 작업이 있으면 SCHED_OTHER 작업보다 먼저 실행될 수 있다.

## 예시
웹 브라우저, 코드 에디터, 일반 백엔드 서버 프로세스는 보통 SCHED_OTHER 범주의 일반 작업으로 실행된다.

## 주의점
- SCHED_OTHER는 실시간 정책이 아니다.
- "기본 정책"이라는 말은 항상 가장 먼저 실행된다는 뜻이 아니다.
- Linux의 실제 내부 구현은 커널 버전에 따라 CFS 또는 EEVDF 흐름으로 이해해야 한다.
