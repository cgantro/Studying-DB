# ISA

## 정의
Instruction Set Architecture. CPU가 제공하는 명령어와 레지스터, 주소 지정 방식의 규약이다.

## 사용 환경
컴파일된 코드가 CPU에서 어떤 명령어와 규칙으로 실행되는지 설명할 때 사용된다.

## 사용 조건
대상 CPU가 해당 ISA를 지원해야 하며, 컴파일러와 어셈블러가 그 규칙에 맞는 기계어를 생성해야 한다.

## 예시
같은 C 코드도 x86으로 컴파일할 때와 ARM으로 컴파일할 때 다른 기계어가 나온다.

## 주의점
- 같은 소스 코드도 ISA가 다르면 생성 기계어가 달라진다.
- 어셈블리는 CPU 종류에 따라 문법과 레지스터가 달라질 수 있다.
- RISC/CISC는 현대 CPU에서 단순 이분법만으로 설명하기 어렵다.
