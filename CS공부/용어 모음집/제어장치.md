# 제어장치

## 정의
명령어를 해석하고 CPU 내부 부품에 제어 신호를 보내는 장치다.

## 사용 환경
명령어를 해석하고 CPU 내부 부품을 움직이는 제어 신호를 만들 때 사용된다.

## 사용 조건
현재 명령어가 IR에 있고 opcode와 상태 플래그를 해석할 수 있어야 한다.

## 예시
load 명령을 만나면 메모리 읽기, 레지스터 쓰기 같은 신호를 만든다.

## 주의점
- 제어장치는 계산 장치가 아니라 지시를 내리는 장치다.
- ALU, 레지스터, 메모리 접근을 조합해 명령어 실행을 만든다.
- 복잡한 명령어일수록 제어 흐름도 복잡해진다.
