# 하이퍼바이저

## 정의
가상머신을 생성하고 하드웨어 자원을 나누어 주는 가상화 계층이다.

## 사용 환경
하나의 물리 시스템에서 여러 격리 실행 환경을 만들거나 배포 단위를 분리할 때 사용된다.

## 사용 조건
가상머신은 하이퍼바이저가 하드웨어 자원을 가상화해야 하고, 컨테이너는 namespace와 cgroup 같은 커널 기능이 필요하다.

## 예시
Type 1 하이퍼바이저는 하드웨어 위에서 직접 VM을 관리한다.

## 주의점
- 가상머신은 게스트 OS를 포함하지만 컨테이너는 호스트 커널을 공유한다.
- 컨테이너 격리는 VM보다 가볍지만 커널 공유에 따른 보안 고려가 있다.
- cgroup은 자원 제한, namespace는 보이는 자원 분리를 담당한다.
