# ALU

## 정의
ALU는 Arithmetic Logic Unit의 줄임말로, 산술 연산과 논리 연산을 수행하는 CPU 내부 실행 장치다.

## 사용 환경
CPU가 add, sub, and, or, compare 같은 연산 명령어를 실행할 때 사용된다. 보통 레지스터에서 값을 읽어 ALU에 넣고, 계산 결과를 다시 레지스터나 플래그에 반영한다.

## 사용 조건
입력 피연산자가 준비되어 있어야 하며, 제어장치가 ALU에 어떤 연산을 수행할지 제어 신호를 보내야 한다.

## 예시
덧셈, 뺄셈, AND, OR, 비교 연산은 ALU를 거쳐 처리된다.

## 주의점
- ALU는 값을 오래 저장하는 공간이 아니라 계산을 수행하는 장치다.
- 어떤 연산을 할지 결정하는 역할은 제어장치가 맡는다.
- 비교 연산 결과는 조건 분기에서 쓰이는 플래그로 이어질 수 있다.
