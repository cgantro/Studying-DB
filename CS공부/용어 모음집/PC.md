# PC

## 정의
Program Counter. 다음에 가져올 명령어의 주소를 저장하는 레지스터다.

## 사용 환경
CPU가 명령어를 실행하는 동안 현재 위치, 중간값, 상태를 빠르게 저장할 때 사용된다.

## 사용 조건
CPU 내부 데이터 경로에서 읽기/쓰기 제어 신호가 맞아야 하며, 문맥 교환 시 필요한 값은 저장되고 복원되어야 한다.

## 예시
분기 명령이 실행되면 PC가 순차 주소가 아니라 분기 대상 주소로 바뀐다.

## 주의점
- 메인 메모리보다 훨씬 빠르지만 용량은 매우 작다.
- 레지스터 값은 실행 흐름의 일부이므로 문맥 교환과 연결된다.
- 레지스터 이름과 개수는 ISA마다 다를 수 있다.
