# 로드맵

## 한 줄 정의
CS 공부를 어떤 순서로 쌓을지 정리한 학습 지도다.

## 핵심 개념
- 기초 → 연결 → 심화 순서로 간다
- 목표는 암기가 아니라 설명 가능 수준이다
- 공부한 내용은 질문으로 바꿔 복습한다

## 세부 정리
- 컴퓨터 구조는 데이터 표현 → CPU/명령어 → 메모리/캐시 → 입출력 흐름으로 잡는다
- 운영체제는 커널/시스템 콜 → 프로세스/스케줄링 → 동기화/메모리/파일/I/O로 잡는다
- 네트워크와 데이터베이스로 웹 시스템 흐름을 연결한다
- 자료구조와 알고리즘은 문제 풀이와 함께 보완한다
- 언어/런타임과 개발 기본기는 실무 기준을 채운다
- 마지막에 면접 대비로 말하기 훈련을 한다

## 예시
- 비트 표현 → CPU → 명령어 → 메모리 → 캐시 → 입출력 흐름을 먼저 이해한다
- 커널/시스템 콜 → 프로세스/스레드 → 스케줄링 → 동기화 → 메모리 관리 → 파일/I/O 흐름으로 운영체제를 묶는다
