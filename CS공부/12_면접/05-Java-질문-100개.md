	```md
	JDK = 개발 도구 + JRE
	JRE = 실행 환경 + JVM
	JVM = 바이트코드를 실제로 실행하는 가상 머신
	
	JVM은 바이트 코드를 실행하는 가상 머신임
	JVM이 바이트코드를 해석 or JIT 컴파일해서 실행함.
	JRE : Java 프로그램을 실행하기 위한 환경(JVM + 기본 라이브러리)
	JDK : Java 개발을 위한 도구 모음, JRE + Javac와 같은 컴파일러 + 디버깅 도구
	```
