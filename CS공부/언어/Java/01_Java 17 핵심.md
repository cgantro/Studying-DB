# Java 17 핵심

## 주요 기능
- `record`: 불변 데이터 전달 객체를 간결하게 표현
- `sealed class`: 상속 허용 범위 제한으로 도메인 모델 안정화
- `pattern matching for instanceof`: 타입 검사 + 캐스팅 보일러플레이트 감소

## 실무 포인트
- DTO/이벤트 모델은 record로 단순화 가능
- 도메인 계층에서 sealed 계층을 쓰면 분기 누락을 줄일 수 있다.
- LTS라서 라이브러리/플랫폼 호환성이 높다.

## 주의점
- 라이브러리 호환성은 빌드 플러그인 및 런타임 이미지와 함께 검증해야 한다.
