## 정의
- Java는 JVM 위에서 동작하는 객체지향 언어다.

## 핵심 개념
- LTS 기준으로 Java 17, 21은 실무 표준으로 많이 사용된다.
- JVM, GC, JIT 이해가 성능/장애 대응에 직접 연결된다.
- Spring Boot와 결합 시 생산성과 생태계 이점이 크다.

## 예시
- 서버 애플리케이션에서 Java 17로 안정 운영 후, Java 21 기능을 점진 도입한다.
