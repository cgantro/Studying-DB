## 🧭 Why LG CNS

> LG CNS에 지원하게 된 계기와 입사 후 포부를 기술해 주세요.
> 

LG CNS에 지원하게 된 계기는 단순히 큰 회사여서가 아니라, ‘기술로 세상을 바꾼다’는 철학을 실제로 구현하고 있다고 느꼈기 때문입니다.

특히 **세종스마트시티** 같은 프로젝트를 보면서, 저도 개발자로서 사람들의 일상을 더 편리하게 만드는 의미 있는 일을 하고 싶다는 생각이 확고해졌습니다.

입사 후 제가 꿈꾸는 모습은 두 가지입니다.

1. 고객이 진짜 필요로 하는 것이 무엇인지 정확히 파악하고, 그것을 **기술로 해결할 수 있는 개발자**가 되고 싶습니다.
2. 언젠가는 사람들에게 “이 시스템 덕분에 정말 편해졌어”라는 말을 들을 수 있는, **실질적인 변화를 만드는 사람**이 되는 것입니다.

## 🛠️ 개발 프로젝트 경험 (Project Experience)

> 본인이 수행한 개발 프로젝트 중 가장 의미 있었던 사례를 구체적으로 기술해 주세요.
> 

팀 프로젝트에서 **해외 거주자에게 한국 간식을 소개하는 플랫폼**의 **백엔드 개발**을 맡았습니다.

사용 기술: **Java 11, Spring Boot, Spring Security, AWS RDS, EC2, Redis**

### 담당 역할

- **사용자 인증·인가 기능 구현**
- **데이터베이스 인프라 설계**
- **성능 테스트 및 개선**

### 주요 구현 내용

- **JWT 기반 인증** 및 **Google OAuth2 소셜 로그인 연동**
- **MySQL 인덱스 최적화**와 **Redis 캐시 도입**으로 DB 부하 감소
- **Spring Security + Redis**를 통한 토큰 만료/갱신 관리 로직 구현
- **k6 부하 테스트 도구**로 시나리오 기반 성능 검증 수행

### 성과

- 로그인 API 평균 응답 속도 **500ms → 150ms로 개선**
- 동시 접속자 50명 이상 환경에서도 **안정적인 인증 처리 유지**
