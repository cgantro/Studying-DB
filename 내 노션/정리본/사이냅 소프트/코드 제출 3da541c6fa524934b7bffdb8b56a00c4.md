## 1. **암아존 배조스씨를 위한 계좌이체 한글 음성 안내**

- **코드**
    
    ```python
    ## 만단위 만, 억 ,조
    unit = ["","만","억","조"]
    name = ["","일","이","삼","사","오","육","칠","팔","구"]
    unit2 = ["", "십", "백", "천"]
    def solve(money):
        answer = ""
        ## 숫자 자릿수 세기
        money= int(money.replace(",","")[:-1])
    
        for i in range(3,-1,-1):
            ## 높은 자리부터
            ten_thousand_unit = 10**(i*4)
            partition = money // ten_thousand_unit
    
            if partition>0:
                answer += makeName(partition,i) + unit[i] + " "
                money %= ten_thousand_unit
    
        return answer[:-1] + "원"
    
    def makeName(partition,unit):
        answer = ""
        num = str(partition)
        ## 천단위
    		# 1000, 999 이런 단위로 값이 들어온다
        print(num)
        length = len(num)
        for i in range(length):
            d = int(num[i])
    				# 0 -> 없음 
            if d > 0:
    						## 1, 10 이상인 경우만 단위만 붙임
                if d == 1 and i != length - 1 :
    						## 1000 -> 4-0-1 = 3 -> 천
                    answer += unit2[length - i - 1]
                else:
    						## 일 이외의 경우 이름 + 단위
                    answer += name[d] + unit2[length - i - 1]
    
        ## 일만일 경우 일 제거
    
    		## 만 단위 unit = 1 일억, 일조 이러지만 일만 이러진 않
        if unit == 1 and answer == "일":
            answer=""
        return answer
    
    with open("input.txt", "r", encoding="utf-8") as file:
        answer = 0
        for line in file:
            amount_str = line.strip()
            korean_representation = solve(amount_str)
            a=list(map(str,korean_representation.split()))
            length = len(a)
            count = 0
            for i in a:
                count+=len(i)
            answer+=count*length
            print(amount_str,korean_representation, answer,length)
        print(answer)
    ```

음성전화에서 일 원 이렇게 음성으로 부르는 게 기억나서 실제로 구현하고 싶었다.

## 2. 스프레드시트 컬럼의 영문명칭 계산

- **코드**
    
    ```python
    dic = {1:'A',2:'B',3:'C',4:'D',5:'E',6:'F',7:'G',8:'H',9:'I',10:'J',11:'K',12:'L',13:'M',14:'N',15:'O',16:'P',17:'Q',18:'R',19:'S',20:'T',21:'U',22:'V',23:'W',24:'X',25:'Y',26:'Z'}
    
    num = int(input())
    result = ''
    ## 26진수
    while num>0:
        ## 나머지로 ABCD를 구한다
        k = num%26
        ## 수가 26으로 딱 나눠지는 경우는 나머지가 0이된다 -> k 26으로 바꿔줌
        if k == 0:
            k =26
        result = dic[k] + result
        num = num//26
    print(result)
    ```

## 3. 숫자 목록을 이용해 만든 두 자연수 합의 최솟값

- **코드**
    
    ```python
    ## 숫자가 정렬 되어 있다
    ## 큰 수가 되도록 작은 자리수에 위치하도록 하나씩 배분하자
    ## 예시
    ## 1,2,3,4,5
    ## 4 5 -> 1의자리 2,3 -> 10의 자리 1-> 100의 자리
    
    ## 0이 있는 경우의 처리
    ## 앞에 넣으면 안된다
    ## -1 조건
    ## 0의 개수k 가 다른 숫자(n - k)들 보다 많을 때
    
    ## 0,0,1,1,2,2,3,3,4
    ## 10034 + 1223 = 11257
    ## 0을 가장 작은 수 다음 자리에 모두 붙여버린 후, 그 다음에 큰 수로 붙여준다
    ## [1,0,1,0,2,2,3,3,4] or [1,1,0,0,2,2,3,3,4] 후자로 바꿔버리자
    
     총 19줄, 452자의 코드[코드 접기]
    
    ## A = A[k:k+2] + A[:k] + A[k+2:]
    def solution():
        A =[0, 0, 1, 8, 2, 2, 8, 9, 0, 3, 4, 0, 0]
        k = A.count(0)
        if k>len(A)-k:
            return -1
        A.sort()
    		## [0, 0, 0, 0, 0, 1, 2, 2, 3, 4, 8, 8, 9]
    		## 0 개수 5 ->  가장 작은 수 두 개 -> 1,2,0,0,0,0,0,2,3,4,8,8,9 나머지
        A = A[k:k+2] + A[:k] + A[k+2:]
        result = 0
        # 자릿수 올려주는 역할
        ten = 1
        ## 끝에있는 원소부터 빼줘야함
        for i in range(len(A)//2):
            result += (A.pop()+A.pop())*ten
            ten*=10
    						##  000248         200389
    		## A= [1] -> 마지막
        ## A의 길이가 홀수일 수 있기 때문에
        result += sum(A)*ten
        return(result)
    
    ```

## 4. **특정 구간내의 모든 피보나치 수의 합**

- **코드**
    
    ```python
    max_value = 99987654321
    min_value = 12345678999
    fibo=[1,1,2,3,5,8]
    ## DP
    
    index = 6
    result = []
    ## max_value보다 작은 값 이내에서 피보나치 수를 구하며 
    ## 구간 조건을 만족한다면 결괏값에 넣는다
    while fibo[-1] <= max_value:
        a = fibo[index-1] + fibo[index-2]
        fibo.append(a)
        if min_value< a <max_value:
            result.append(a)
        index+=1
    
    print(sum(result))
    ```
