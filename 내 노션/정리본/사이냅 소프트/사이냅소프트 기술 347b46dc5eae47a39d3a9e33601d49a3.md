# 사이냅소프트 기술

1. **사이냅 문서뷰어 2022:**
    - 다양한 문서 형식 지원 (PDF, Word, Excel 등)
    - 화면 확대/축소 및 회전 기능
    - 주석 및 메모 추가 기능
2. **사이냅 문서필터:**
    - 다양한 문서 포맷 변환 지원
    - 문서 압축 및 암호화 기능
    - OCR(광학 문자 인식) 지원
3. **사이냅 에디터:**
    - 다양한 문서 편집 기능 (텍스트, 이미지, 표 등)
    - PDF 편집 및 변환 기능
    - 실시간 공동 작업 기능
4. **사이냅 OCR Pro:**
    - 광학 문자 인식을 통한 텍스트 추출 기능
    - 이미지 및 스캔된 문서에서 텍스트를 추출하여 편집 가능
5. **사이냅 오피스:**
    - 워드, 엑셀, 파워포인트와 같은 오피스 문서 생성 및 편집 기능
    - 다양한 문서 형식 지원
6. **사이냅 폼:**
    - 양식 디자인 및 작성 기능
    - 전자 서명 및 양식 제출 기능
7. **사이냅 피디포커스:**
    - 프레젠테이션 및 문서의 시각적 효과 부여 기능
    - 다양한 템플릿 및 디자인 요소 제공
