# 웹 기반 스마트 포트폴리오 & 개인비서 서비스

진행 상태: 완료됨
시작일: 2025년 8월 7일
종료일: 2025년 8월 20일
기술 스택: CSS3, HTML, JavaScript

## **📁 프로젝트 개요**

HTML/CSS/JavaScript 기반으로 제작된 웹 프로젝트로,

사용자의 포트폴리오 제공, 간단한 내비게이션, 날씨 조회, 챗봇, 음성 기능 등을 포함하는 **종합 개인 웹 서비스**입니다.

---

# ✅ 1. 기술 스택

### **Frontend**

- **HTML5**, **CSS3**, **JavaScript (ES6+)**
- 반응형 레이아웃 구성 (tablet-layout.css)
- DOM 기반 화면 전환 및 UI 인터랙션 구현

### **API / 기능**

- **Weather API** 사용 (추정: 날씨 조회 기능 존재)
- **Navigator API** 기반 위치 정보 활용
- **Speech Recognition / Speech Synthesis API**(웹 음성 기능, voice.js 존재)
- 간단한 맵/내비게이션 스크립트(map.js, navigation.js)

### **환경 / 기타**

- 로컬 환경에서 정적 웹 서비스 형태로 개발
- Asset 최적화 (이미지, css, script 분리)

---

# ✅ 2. 주요 기능

### ✅ **1. 개인 포트폴리오 페이지**

- portfolio.html + portfolio.js
- 프로젝트 정보/이미지를 동적으로 로딩
- portfolio-data.js를 활용해 유지보수성 강화
- 사용자 정보, 서비스 소개, 이미지 갤러리 구현

---

### ✅ **2. 챗봇 기능**

- chatbot.html + chatbot.js
- 사용자 입력 기반 기본 rule-based 챗봇 구현
- UI 구성 및 메시지 입출력 처리

---

### ✅ **3. 날씨 서비스**

- weather.html + weather.js
- Weather API 요청 후 현재 날씨 및 지역 정보 표시
- 에러 처리 및 위치 권한 확인 로직 포함

---

### ✅ **4. 내비게이션/지도**

- navigation.html + navigation.js
- 지도 이미지 기반 간단한 경로 안내
- 사용자 위치 정보 또는 선택된 경로 표시

---

### ✅ **5. 음성 인터랙션**

- voice.js
- Web Speech API 이용해
    - 음성 명령 인식
    - 음성 안내 출력
- 버튼 클릭으로 음성 활성화 가능

---

### ✅ **6. 유저 로그 시스템**

- log.html + log.js
- 사용자의 행동/로그 기록을 화면에서 확인할 수 있는 기능

# ✅ 4. 배운 점 (이력서용 문장)

- 단순 정적 페이지가 아니라 **여러 독립된 기능을 가진 서비스 구조를 설계**하며 페이지 간 자원 분리 및 모듈화를 학습함.
- JavaScript로 **DOM 제어, 상태 관리, 이벤트 처리** 등 기초 웹 개발 역량을 강화함.
- Weather API / Speech API 등 **브라우저 내장 API 및 외부 API 연동 경험**을 쌓아 기능 확장 방법을 익힘.
- 구조화된 프로젝트 구성(styles, scripts, pages 분리)을 통해 **확장 가능한 웹 아키텍처 설계 능력**을 향상함.
- 포트폴리오 데이터(프로젝트 정보)를 별도 JS 파일로 관리하여 **데이터 중심 UI 설계 패턴**을 경험함.
