- 면접준비>
    
    DRM,DLP에 대한 기초만 안다. -> 구체적인 건 몰라도, 이론적인 건 공부를 하고 가야한다.
    커뮤니케이션 관련
    
    자기소개, 하루 일과, 회사에서 무엇을 얻어가고 싶은지, 정규화, GC, 이력서에 나와있는 내용에 대한 질문, 회사에 대해 궁금한 점.
    JobPlanet에 있는 내용들 싹다 정리해서 정리하기
    
    1. 리눅스 명령어에 대해서 좀 공부를 해가기
    2. Java 문법, 이론 등을 글로 정리해서
    3. 과제 수행하면서 문제 어떻게 해결했는지.
    4. 회사 관련, 가치관 등 리스트업.
    5. 기업에 오고 싶다라는 느낌을 줘야함.
    6. 답변 준비 싹 다 글로 정리
    7. 인성 질문 : 가치관 (회사의 인재상 or 날 것 그대로의 가치관 : 성장(구체적인 사례: 끝까지 알기위해 어떤 노력을 했나) -> 공식문서)
    8. 장&단점 :
        1. 성격적 장점(책임감) -> 사례?, 또한 성장과 연결)
        2. 기한에 맞추기 위해서 내가 원하는 퀄리티는 아니어도 넘어가려고 한다.
        (완성도를 높이려고 하지만 팀프로젝트에서 좋지많은 않다. -> 일단 완성 후 퀄리티를 높이려는 시도를 한다.)
    9. 항상 구체적으로 말하기.
    10. 소중하게 여기는 가치: 성장, 가족(더불어 함께!)
    
    항상 모든 것은 사례.
    
- 면접 티
    
    직접적인 말보다 뉘앙스(큰 그림, 개념을 전달하는 느낌)

    동작 신뢰성, 응답성, 보안성
